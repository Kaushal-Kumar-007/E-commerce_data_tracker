# 🛒 Madhav Ecommerce Sales Dashboard – Power BI Project

## 📌 Project Overview
This project is an interactive **Ecommerce Sales Dashboard** created in **Power BI** to analyze sales performance, customer behavior, profit trends, and payment insights across different states and product categories.

The dashboard helps businesses track important KPIs such as:
- Total Sales Amount
- Total Profit
- Quantity Sold
- Average Order Value (AOV)
- Monthly Profit & Loss Trends
- Customer Purchase Analysis
- Category & Sub-Category Performance
- Payment Mode Distribution

# 📂 Dataset Information

The project uses two datasets:

## 1️⃣ Orders Table
Contains customer and order-related information.

| Column Name | Description |
|---|---|
| Order ID | Unique order identifier |
| Order Date | Date of order |
| CustomerName | Name of customer |
| State | Customer state |
| City | Customer city |

---

## 2️⃣ Details Table
Contains transactional and product-related information.

| Column Name | Description |
|---|---|
| Order ID | Unique order identifier |
| Amount | Sales amount |
| Profit | Profit earned/loss |
| Quantity | Quantity purchased |
| Category | Product category |
| Sub-Category | Product sub-category |
| PaymentMode | Payment method used |
| AOV | Average Order Value |

---

# 🔄 Data Cleaning & Transformation (Power Query)

The following transformations were performed using **Power Query**:

- Removed null and duplicate values
- Checked data types for accuracy
- Formatted date columns
- Created relationships between tables using `Order ID`
- Cleaned categorical data
- Optimized dataset for reporting

---

# 🧮 DAX Measures Used

```DAX
Total Sales = SUM(Details[Amount])

Total Profit = SUM(Details[Profit])

Total Quantity = SUM(Details[Quantity])

Average Order Value = AVERAGE(Details[AOV])

Profit Percentage = 
DIVIDE(SUM(Details[Profit]), SUM(Details[Amount])) * 100
```
